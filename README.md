# 2048: Imperial China Edition
A historical / educational version of Gabriele Cirulli's [original 2048](http://gabrielecirulli.github.io/2048/). 

The original version is a small clone of [1024](https://play.google.com/store/apps/details?id=com.veewo.a1024), based on [Saming's 2048](http://saming.fr/p/2048/) (also a clone).

### Notes for the Imperial China Version
This is not an all-inclusive view of Chinese history but a brief overview with the highlights. Dates referenced from [Wikipedia](http://en.wikipedia.org/wiki/History_of_China) as of April 29, 2014.
